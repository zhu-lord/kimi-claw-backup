# 即梦 Seedance 2.0 提示词构建工具

通过引导式对话，帮助你逐步构建高质量、高通过率的视频提示词，基于Seedance 2.0安全规范，提供结构化提示词模板。

## 何时使用此 Skill

**必须**在以下情况下使用此 skill：
- 用户想要生成即梦 Seedance 2.0 视频提示词
- 用户需要创建 AI 视频生成提示词
- 用户想要优化视频描述/脚本
- 用户询问如何写视频提示词
- 用户提到"Seedance"、"即梦"、"视频生成"、"提示词"

**不要**直接回答用户，必须使用此 skill 的引导式流程。

## 工作流程（必须严格遵循）

### 核心原则
1. **不要一次性输出所有内容**
2. **必须**通过多轮对话收集信息，逐问以下步骤
3. **每轮只问当前步骤的问题**，不要提前问后续步骤
4. 收集完所有信息后，按"输出格式"生成最终提示词

### 七步构建流程

#### 第一步：素材准备
询问用户：
- 文字提示词：详细描述你想要的内容
- 首帧/尾帧图片（最多9张）：每张图片的具体用途和特点
- 参考视频（最多3个）：每个视频的参考点（运镜/动作/风格等）
- 背景音乐/音效（最多3个）：每个音频的用途和风格

#### 第二步：核心内容
询问用户：
- 角色：外貌、服装、发型、性格特点
- 场景：地点、环境细节、时间、天气、光线
- 动作：具体动作、姿势、表情、情绪变化
- 互动：角色之间的关系和互动方式

#### 第三步：类型细化
根据内容类型，进一步询问：

| 类型 | 追问内容 |
|------|----------|
| 剧情叙事 | 详细对话/台词、情绪变化曲线、分段时间安排、角色动机 |
| 产品展示 | 具体卖点、展示角度和方式、产品细节、使用场景 |
| 动作打斗 | 角色数量和特点、动作风格和招式、氛围和节奏、胜负结果、具体攻防拆招动作、每个镜头时长 |
| 风光风景 | 具体场景类型、时间/天气变化、光线效果、季节特点 |
| 创意特效 | 特效类型和效果、变身起止状态、视觉冲击点、风格融合 |

#### 第四步：镜头语言
询问用户：
- 镜头运动：固定/推进/拉远/摇/跟拍/环绕/升降（具体速度和幅度）
- 景别：远景/全景/中景/近景/特写（具体距离和范围）
- 角度：平视/仰视/俯视/低角度（具体角度数值）
- 镜头特性：焦距、景深、跟随方式、焦点变化
- 是否一镜到底：如果是，具体拍摄路线

#### 第五步：声音音效
询问用户：
- 背景配乐：风格、节奏、音量、情绪
- 环境音效：具体环境音、声音来源、音量变化
- 对话/旁白：具体台词内容、语气、音色、音量
- 动作音效：具体动作的声音效果、音量和时机

#### 第六步：时间节奏
询问用户：
- 视频总时长：精确到秒
- 分时间段描述：每个时间段的具体内容和时长
- 节奏变化：快慢节奏的转换点和方式
- 重点内容：需要强调的关键画面和时间点

#### 第七步：风格确认
询问用户：
- 视频风格：电影级写实/动漫卡通/商业广告/创意实验（具体参考作品）
- 画面比例：16:9/2.35:1/9:16/1:1
- 色调：整体色彩倾向、饱和度、对比度
- 光影：光线类型、强度、方向、阴影效果
- 质感：画面质感、细节表现、特效风格
- 特殊要求：胶片颗粒感/真实光线/视觉冲击/其他特殊效果

## 输出格式

收集完所有信息后，按以下结构输出最终提示词：

```
【安全声明】
[安全声明词，如：原创数字艺术作品/积极正向主题/适合全年龄段观看/无版权争议/无敏感元素/虚构场景]

[素材指定区]
@图片1作为[具体用途，如：首帧/角色形象/场景参考]
@视频1作为[具体用途，如：运镜参考/动作参考/风格参考]
@音频1作为[具体用途，如：背景音乐/环境音效/台词参考]

[时间线区]
(00-1.5s):[镜头运镜]+[环境光影]+[主体起势动作与微表情]。
(1.5-3.0s):[运镜变化]+[主体动态发展/位移]+[物理环境互动]。
(3.0-4.0s):[冲击力运镜]+[动作高潮/关键帧]+[面部/细节特写]。
(4.0-5.5s):[收尾运镜]+[动作惯性/意外转折]+[环境余韵]。
...

**注意**：每个时间段占用一行，确保格式清晰易读。

[风格区]
[整体风格详细描述，包括色调、光影、质感、特效风格、画面比例、帧率]，[风格关键词，如：Cinematic/8K/电影感/写实主义/奇幻风格]

[安全检查区]
- 适合全年龄段观看
- 无版权争议
- 无敏感元素
- 虚构场景
- 原创内容
```

## 核心指令

- **禁止废话**：输出中禁止出现"画面基调"、"光影设定"、"分析思路"等标题。不要输出任何开场白或结束语。
- **内容融合**：将光影、氛围、镜头语言全部融入到每一秒的动作描述中，而不是单独列出。
- **直接输出**：直接输出带有时间标记的提示词段落，格式必须便于直接复制使用。

## 风格指南

### 写实风格
电影级写实风格，2.35:1宽银幕，24fps，真实胶片颗粒，自然色彩，自然光源，真实阴影，柔和过渡。

### 创意风格
梦幻画面，华丽特效，视觉冲击力强，大胆色彩，创意光源，色彩变换，光影对比强烈。

### 广告风格
商业化的摄像展示，特写镜头，产品细节展示，明亮色彩，均匀布光，产品突出，质感表现。

### 动漫卡通风格
卡通化形象，鲜明色彩，线条清晰，风格统一，扁平化光线，色彩明快，对比强烈。

### 实验风格
创新构图，非常规视角，艺术化处理，独特风格，实验性 lighting，光影艺术，创意照明。
