# 高频战斗词汇库

## 一、连续攻击词汇

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **连续出拳** | 拳头连续击出，每拳之间无停顿，左右交替或同拳连击 | `左右拳交替击出，左刺拳接右直拳接左勾拳，三连击无间隔` |
| **连环拳** | 同一手连续击出多拳，或双手交替形成拳法连击 | `右拳连续三击，先直拳后勾拳再摆拳，连环拳攻击对手头腹` |
| **连续踢腿** | 腿法连续击出，一腿未落另一腿已起，或同腿连踢 | `左腿低扫接右腿中扫接左腿高扫，连续三腿踢击不同高度` |
| **连环腿** | 同一腿连续踢击，或双腿交替踢出 | `右腿连续两记鞭腿，先踢大腿外侧再踢头部，连环腿攻击` |
| **拳腿连击** | 拳法与腿法无缝衔接，形成立体攻势 | `左刺拳接右直拳接左低扫腿，拳腿连击三招一体` |
| **组合连击** | 多种技法组合连续攻击，包含拳、腿、肘、膝 | `左刺拳接右勾拳接左横肘接右正膝，四招组合连击` |

## 二、攻防转换词汇

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **攻防转换** | 攻击与防守切换，无停顿转换 | `左拳击出即收，右手同时格挡对手反击，攻防转换` |
| **攻防一体** | 攻击动作同时具备防守功能，一动作两用 | `左臂上架格挡同时右拳击出，攻防一体` |
| **瞬防瞬攻** | 防守后瞬间转为攻击，零延迟反击 | `左手拍挡后0.1秒右拳击出，瞬防瞬攻零间隔` |
| **闪击** | 闪避同时发动攻击，躲避与进攻同步 | `身体左侧闪同时右拳击出，闪击对手面部` |
| **截击** | 在对手攻击发动时抢先攻击，截断对手攻势 | `对手起腿瞬间右直拳截击其面部，截断攻势` |
| **迎击** | 在对手攻击过程中迎面反击 | `对手摆拳来袭时右直拳迎击其面部，迎击得手` |

| **后撤** | 身体后移拉开距离 | `后脚蹬地，身体后撤半步，避开对手踢击` |
| **提膝防守** | 膝盖提起阻挡低段攻击 | `右腿提膝，小腿竖直阻挡对手低扫腿` |
| **双臂护头** | 双臂抬起保护头部 | `双臂抬起护住头部两侧，形成防守姿态` |
| **封堵** | 手臂封堵对手攻击路线 | `右臂下压封堵对手勾拳路线，阻断攻势` |
| **滑步闪避** | 步法移动躲避攻击 | `左脚滑步向左前方，身体侧移躲过对手直拳` |
| **摇闪** | 身体弧形摆动躲避连续攻击 | `身体以腰为轴弧形摇闪，头部U形轨迹躲过两记直拳` |
| **搂抱** | 近身搂抱限制对手攻击 | `对手连击时近身，双臂搂抱对手限制出拳` |
| **撤步防守** | 后撤步拉开距离防守 | `右脚后撤一步，身体后移避开对手攻击范围` |
| **转身闪避** | 身体转身躲避攻击 | `身体右转，侧身躲过对手直拳` |
| **低位防守** | 手臂下压防守低段攻击 | `左臂下压，格开对手低扫腿` |
| **高位防守** | 手臂上抬防守高位攻击 | `右臂上抬，格挡对手高鞭腿` |

## 四、躲闪技巧词汇

**躲闪核心原则**：通过身体位移避开攻击，而非接触性格挡，为反击创造机会。

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **侧闪** | 头部和躯干向侧面移动，避开直线攻击 | `头部和躯干向左侧移，躲过对手右直拳，左侧闪` |
| **下潜** | 双膝弯曲，身体重心下降，躲避高位攻击 | `双膝弯曲，身体下沉，躲过对手摆拳，下潜` |
| **后仰** | 上身后仰，头部后撤，避开前向攻击 | `上身后仰重心后移，头部后撤，避开对手直拳` |
| **摇闪** | 身体以腰部为轴弧形摆动，头部呈U形轨迹 | `身体以腰为轴弧形摆动，头部U形轨迹躲过两记直拳` |
| **滑闪** | 步法滑动配合身体侧移，整体位移躲避 | `左脚滑步左前方，身体侧移，滑闪躲过直拳` |
| **蹲闪** | 深蹲躲避高位攻击，同时准备反击 | `身体深蹲，躲过对手高鞭腿，蹲闪蓄力` |
| **跳闪** | 小跳步后撤或侧移，拉开距离躲避 | `双脚小跳后撤，避开对手扫腿，跳闪` |
| **转闪** | 身体转动带动躲闪，侧身避开攻击 | `身体右转90度，侧身躲过对手直拳，转闪` |
| **折闪** | 上身前折叠后仰，躲避头部攻击 | `上身前折叠后仰，躲过对手摆拳，折闪` |
| **撤闪** | 后撤步配合身体后移，整体后退躲避 | `右脚后撤一步，身体后移，撤闪避开踢击` |
| **提肩闪** | 单肩提起护住下颌，头部侧偏躲避 | `左肩提起护住下颌，头部右偏，提肩闪躲过直拳` |
| **缩颈闪** | 颈部收缩，头部下沉躲避高位攻击 | `颈部收缩，头部下沉，缩颈闪躲过摆拳` |
| **侧身闪** | 身体侧转缩小受击面积，避开攻击 | `身体左转侧身，缩小受击面，侧身闪躲过直拳` |
| **弧形闪** | 身体沿弧线移动躲避，同时调整位置 | `身体沿弧线向左移动，弧形闪躲过攻击并调整站位` |
| **V形闪** | 身体先向一侧再向另一侧移动，呈V形轨迹 | `身体先左闪再右移，V形闪躲过连续两拳` |

## 五、躲闪组合技巧

| 组合名称 | 动作序列 | 提示词示例 |
|---------|---------|-----------|
| **连续侧闪** | 连续左右侧闪躲避连击 | `头部先左侧闪躲过右直拳，随即右侧闪躲过左摆拳，连续侧闪` |
| **下潜起身** | 下潜后起身反击 | `下潜躲过摆拳，起身同时右勾拳击打对手腹部` |
| **摇闪反击** | 摇闪后立即反击 | `摇闪躲过两记直拳，第三拳时右直拳反击` |
| **闪退组合** | 闪避后撤拉开距离 | `左侧闪躲过直拳，接后撤步拉开距离` |
| **闪进组合** | 闪避同时前进缩短距离 | `右侧闪同时滑步前进，闪进至对手侧翼` |
| **三连闪** | 连续三次躲闪动作 | `左侧闪接下潜接后仰，三连闪躲过三连击` |

## 七、匕首攻击词汇

**匕首持握方式：**

| 持握方式 | 描述 | 特点 |
|---------|------|------|
| **正握** | 刀刃向上，刀尖向前，握柄于掌心 | 适合刺击、挑击 |
| **反握** | 刀刃向下，刀尖向后，握柄于掌心 | 适合划割、格挡反击 |
| **冰锥握** | 刀尖向下，刀刃向外，反手握持 | 适合下刺、隐蔽攻击 |

**匕首攻击动作：**

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **直刺** | 匕首直线向前刺出，攻击要害 | `右手正握匕首直线刺向对手腹部，直刺攻击` |
| **上挑** | 匕首由下向上弧形挑击 | `匕首由下向上挑击对手下颌，上挑攻击` |
| **下刺** | 匕首由上向下刺击 | `右手反握匕首由上向下刺向对手锁骨，下刺攻击` |
| **横划** | 匕首横向划割 | `匕首从左向右横划对手颈部，横划攻击` |
| **反撩** | 匕首由下向后上方弧形撩击 | `匕首由下向后上方撩击对手手臂内侧，反撩` |
| **斜劈** | 匕首沿斜线劈下 | `匕首从右上向左下斜劈对手肩部，斜劈攻击` |
| **连刺** | 匕首连续刺击 | `匕首连续三次刺向对手胸腹，连刺三击` |
| **旋刺** | 身体旋转配合刺击 | `身体右转180度，右手匕首顺势刺向身后对手，旋刺` |
| **双持交叉** | 双匕首交叉攻击 | `左右手匕首交叉刺向对手胸腹两侧，双持交叉攻击` |
| **双持连击** | 双匕首交替连续攻击 | `左手匕首刺出收回，右手匕首随即刺出，双持连击` |
| **背刺** | 从背后刺击要害 | `绕至对手身后，匕首刺向背部要害，背刺` |
| **喉刺** | 刺向咽喉要害 | `匕首直线刺向对手咽喉，喉刺攻击` |
| **腕划** | 划割对手手腕 | `匕首横划对手持械手腕，腕划缴械` |
| **腿划** | 划割对手腿部 | `匕首划割对手大腿内侧，腿划攻击` |
| **飞刀** | 投掷匕首攻击 | `右手甩出匕首，飞刀直射对手胸部` |

**匕首组合攻击：**

| 组合名称 | 动作序列 | 提示词示例 |
|---------|---------|-----------|
| **刺挑组合** | 直刺接上挑 | `匕首直刺腹部，随即上挑下颌，刺挑组合` |
| **划刺组合** | 横划接直刺 | `匕首横划颈部，对手后仰时直刺胸腹，划刺组合` |
| **双持四连** | 双匕首四连击 | `左刺右刺左划右挑，双持四连击` |
| **旋身连刺** | 旋转配合连刺 | `身体左转匕首刺向左侧，继续旋转刺向后方，旋身连刺` |
| **下潜刺** | 下潜后刺击 | `下潜躲过攻击，起身同时匕首上挑对手腹部` |
| **闪身刺** | 闪避同时刺击 | `左侧闪躲过直拳，右手匕首刺向对手肋部` |

## 八、节奏控制词汇

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **高频攻击** | 动作频率高，每秒3-5个动作 | `高频攻击连续出拳，每秒四拳，左右交替直拳连击` |
| **紧凑连击** | 动作紧凑，攻击密度高 | `紧凑连击五拳，左刺右直左勾右摆左劈，一气呵成` |
| **高频互攻** | 双方交替攻防，节奏紧凑 | `双方高频互攻，每秒各出三招，攻防交替` |
| **连续压制** | 持续攻击不给对手喘息机会 | `连续压制八拳，不给对手防守间隙，步步紧逼` |
| **爆发连击** | 短时间内高强度连续攻击 | `爆发连击六招，两秒内完成三拳两腿一膝` |
| **密集攻击** | 攻击频率高，招式密集 | `密集攻击，三秒内十拳连击覆盖对手头腹` |

## 十一、无缝衔接词汇

| 词汇 | 动作要领 | 提示词示例 |
|------|---------|-----------|
| **无缝衔接** | 前一动作结束即后一动作开始，无停顿 | `左拳收回同时右拳击出，无缝衔接` |
| **一气呵成** | 连续动作流畅完整，无断裂 | `五招连击一气呵成，中间无停顿` |
| **流畅连击** | 连击动作流畅自然 | `三拳两腿流畅连击，动作行云流水` |
| **连贯攻击** | 攻击动作连贯不断 | `连贯攻击六招，招招相连` |
| **零间隔** | 动作之间无时间间隔 | `三拳零间隔连击，每拳间隔小于0.1秒` |
| **无停顿** | 动作持续无停顿 | `无停顿连续出拳八次` |

## 十二、高频战斗场景示例

**示例1：高频对攻**
```
双方高频互攻，每秒各出三招，左刺拳对右直拳，右勾拳对左格挡，左低扫对右提膝，攻防交替，三秒内共十八个动作
```

**示例2：连续压制**
```
连续压制八招：左刺拳→右直拳→左勾拳→右摆拳→左低扫→右中扫→左横肘→右正膝，每招间隔0.2秒，一气呵成
```

**示例3：爆发连击**
```
爆发连击六招：两秒内完成左刺拳→右直拳→左勾拳→右鞭腿→左横肘→右正膝，无缝衔接无停顿
```

**示例4：瞬时攻防**
```
瞬时攻防转换：左手拍挡后0.1秒右直拳反击，对手格挡后0.1秒左勾拳再攻，双方每秒各完成四次攻防转换
```

**示例5：密集攻击**
```
密集攻击十连击：三秒内六拳三腿一膝，左刺右直左勾右摆左劈右贯，接左低扫右中扫左高扫，最后右膝顶击，招招相连零间隔
```
