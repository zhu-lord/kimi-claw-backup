# 受击状态描写与检查清单

## 受击状态描写（仅在用户明确要求时使用）

**⚠️ 重要使用规则**：
- **仅在用户明确要求时添加受击/受伤描写**
- 用户未提示时，不主动添加任何受击、受伤、流血等描写
- 即使战斗场景激烈，也不擅自加入伤势描写
- 保持提示词灵活性，适用于不同风格的战斗场景需求，适当时机可以给受伤处特写

**核心原则**：使用中性、客观的描述，避免血腥暴力词汇，保持适合全年龄段观看。

## 五、伤势程度分级

| 程度 | 描述特点 | 提示词示例 |
|------|---------|-----------|
| **轻微** | 表皮伤、无出血或微量渗出 | `皮肤表面轻微擦伤，泛红无出血` |
| **轻度** | 浅表伤、少量渗出 | `浅表划伤，有少许渗出，周围皮肤轻微红肿` |
| **中度** | 明显淤青、肿胀 | `硬币大小淤青，颜色青紫，轻微肿胀` |
| **明显** | 多处伤势、影响动作 | `面部两处淤青，嘴角轻微裂开，呼吸略显急促` |

## 七、安全词汇对照表

| 避免使用 | 安全替代 |
|---------|---------|
| 鲜血、血流如注、血肉模糊 | 渗出、少量液体、皮肤破损 |
| 伤口撕裂、皮开肉绽 | 皮肤裂开、表皮破损 |
| 骨折、断骨、碎裂 | 骨骼受损、活动受限 |
| 痛苦、惨叫、哀嚎 | 表情紧绷、呼吸急促、眉头紧锁 |
| 大量出血、血泊 | 渗出、液体流出 |
| 伤口深可见骨 | 深层伤势、伤及皮下组织 |

## 八、受伤处特写使用规则

- **根据战斗实际情况决定是否给特写**：非所有受伤都需要特写
- **重要伤势给特写**：影响战斗节奏、角色状态变化、剧情转折的伤势应给特写
- **轻微伤势可不给特写**：不影响战斗的轻微擦伤可融入动作描写，无需单独特写
- **特写时机合理**：在动作间隙、节奏转换、双方拉开距离时给特写，避免打断战斗流畅性
- **特写时长控制**：特写时长通常0.5-1秒，不宜过长影响节奏
- **特写镜头语言**：使用推近、切镜等方式，配合光线、角度突出伤势细节

**推荐做法**：
- 伤势描写与动作描写结合
- 通过伤势体现战斗强度
- 通过角色反应体现意志品质
- 保持积极正向的主题基调
- 伤势不影响角色继续战斗的能力

**禁止内容**：
- 深度伤口、开放性骨折等严重伤势
- 大量出血、动脉出血等详细描述
- 剧痛、惨叫等极端反应
- 伤势导致角色丧失战斗能力
- 任何可能引发不适的详细描写

## 九、受击状态描写示例

**示例1：面部受击**
```
左直拳击中右侧颧骨，皮肤瞬间泛红，0.5秒后出现硬币大小红肿。嘴角轻微裂开，有少许渗出沿下颌滑落。眉头紧锁，呼吸节奏短暂紊乱。
```

**示例2：躯干受击**
```
右膝顶击腹部左侧，皮肤表面出现掌心大小淤青，颜色由红渐转淡紫。躯干肌肉紧绷，呼吸时腹部轻微起伏受限。
```

**示例3：连续受击**
```
连续三拳击中面部：左眼眶周围出现淡青色淤痕，右侧颧骨红肿，嘴角轻微裂开。额头渗出沿眉骨滑向太阳穴。眼神依然坚定，呼吸节奏逐渐恢复。
```

**示例4：腿部受击**
```
低扫踢击中右大腿外侧，皮肤表面出现椭圆形淤青，面积约半个掌心。右腿支撑时肌肉轻微颤抖，步态略有调整。
```

**示例5：持久战伤势累积**
```
战斗进行至中段：左眼眶下方青紫色淤青明显，右侧嘴角裂开，右前臂外侧一道3厘米浅表划伤。额头渗出沿鬓角滑落。呼吸急促但节奏稳定，眼神依然锐利。
```

## 十、伤势与动作关联

| 伤势部位 | 动作影响 | 提示词示例 |
|---------|---------|-----------|
| **眼部淤青** | 视线受阻、眨眼频率增加 | `左眼眶淤青肿胀，眨眼频率增加，头部微侧调整视线` |
| **嘴角裂开** | 表情受限、呼吸调整 | `嘴角裂开，表情变化时肌肉轻微牵拉，呼吸通过鼻腔` |
| **肋部伤势** | 躯干活动受限 | `左侧肋部淤青，躯干左转时动作幅度减小` |
| **腿部伤势** | 步法调整、重心变化 | `右大腿淤青，重心略微偏向左腿，步幅减小` |
| **手臂伤势** | 出拳速度和力量变化 | `左前臂擦伤，左拳击出速度略微降低，右手护防加强` |

- [ ] 零形容词：未使用"猛烈"、"快速"、"华丽"等修饰语

- [ ] 动作连贯性：新动作与上一镜头结尾体态和位置无缝衔接
- [ ] 符合人体工学：动作逻辑合理，符合物理规律

### 高频率打戏特殊检查项
- [ ] 连续组合动作：设计3-5招的连续组合，动作间无缝衔接
- [ ] 攻防同步：描述双方同时动作的细节
- [ ] 镜头技巧：使用快速跟拍、摇镜头、视角切换和慢动作插入
- [ ] 节奏控制：使用0.5-1秒的短时间区间，包含短暂停顿
- [ ] 物理反馈：详细描述连续反应、平衡破坏、肌肉张力和皮肤与衣物变化

### 受击状态描写检查项
- [ ] **使用时机确认**：仅在用户明确要求时添加受击/受伤描写，用户未提示时不主动添加
- [ ] 伤势程度适当：仅描述表皮伤、浅表伤、淤青、肿胀等轻微伤势
- [ ] 词汇安全合规：使用中性、客观的描述，避免血腥暴力词汇
- [ ] 出血描写适度：使用"渗出"、"少量液体"等安全词汇，避免详细出血描述
- [ ] 伤势与动作关联：伤势描写与动作描写结合，体现战斗强度
- [ ] 角色反应合理：通过表情、呼吸、眼神等体现角色状态，避免极端反应
- [ ] 保持战斗能力：伤势不影响角色继续战斗的能力
- [ ] 符合全年龄段标准：确保伤势描写适合全年龄段观看
- [ ] 积极正向主题：通过伤势体现意志品质，保持积极正向基调
