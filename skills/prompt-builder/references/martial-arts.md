# 武术动作库

## 一、步法类动作

### 1. 虚步
**动作要领**：后腿屈膝半蹲，脚尖外展45度，全脚掌着地；前腿微屈，脚跟离地，脚尖虚点地面，身体重心落于后腿。
**提示词示例**：`右腿屈膝半蹲全脚掌着地，左脚尖虚点地面，重心置于右腿，呈右虚步站立`

### 2. 马步
**动作要领**：双脚开立约三脚宽，脚尖正对前方，屈膝半蹲，膝部不超过脚尖，大腿接近水平，挺胸塌腰。
**提示词示例**：`双脚开立宽于肩，屈膝半蹲大腿近水平，挺胸塌腰呈马步`

### 3. 弓步
**动作要领**：前腿屈膝前弓，膝部不超过脚尖，后腿挺膝伸直，脚尖内扣，两脚全脚掌着地。
**提示词示例**：`左腿屈膝前弓，右腿挺膝伸直脚尖内扣，呈左弓步`

### 4. 仆步
**动作要领**：一腿屈膝全蹲，另一腿伸直平仆于地面，脚尖内扣，两脚全脚掌着地。
**提示词示例**：`右腿屈膝全蹲，左腿伸直平仆地面脚尖内扣，呈右仆步`

### 5. 歇步
**动作要领**：两腿交叉屈膝全蹲，前脚全脚掌着地，后脚脚跟离地，臀部坐于后腿小腿上。
**提示词示例**：`两腿交叉屈膝全蹲，臀部坐于后腿，呈歇步`

### 6. 独立步
**动作要领**：一腿站立支撑，另一腿屈膝提起，脚尖自然下垂，大腿高于水平。
**提示词示例**：`右腿独立支撑，左腿屈膝提起脚尖下垂，呈右独立步`

### 7. 垫步
**动作要领**：后脚向前脚并步，同时前脚提起前移，快速缩短与对手距离。
**提示词示例**：`右脚向左脚并步，左脚同时提起前移，垫步逼近`

### 8. 滑步
**动作要领**：前脚先向前滑出一步，后脚随即跟上，保持两脚原有距离，重心平稳移动。
**提示词示例**：`左脚向前滑出，右脚随即跟上，滑步前移`

## 二、踢腿类动作

### 1. 变线踢
**动作要领**：先做高踢假动作，在对手防守时迅速改变踢击路线，转向低位或其他角度攻击。常见为高位转低位、直线转弧线。
**提示词示例**：`右腿先向高位摆出，中途迅速下压转为低扫踢击对手大腿外侧，变线踢`

### 2. 侧踢
**动作要领**：身体侧对目标，踢击腿屈膝提起，脚尖勾起，以脚外侧或脚跟为力点，直线蹬出。
**提示词示例**：`身体右转侧对目标，左腿屈膝提起脚尖勾起，以脚跟直线蹬出，左侧踢`

### 3. 扫腿
**动作要领**：以支撑腿为轴，另一腿伸直由外向内弧形扫击对手下盘，攻击脚踝或小腿。
**提示词示例**：`以右腿为轴，左腿伸直由外向内弧形扫击对手脚踝，低扫腿`

### 4. 正蹬
**动作要领**：正面面对目标，踢击腿屈膝提起，脚尖勾起，以脚跟为力点直线向前蹬出。
**提示词示例**：`右腿屈膝提起脚尖勾起，以脚跟向前直线蹬出，正蹬踢`

### 5. 后摆腿
**动作要领**：身体背对目标，踢击腿伸直向后上方摆动，以脚跟或脚底为力点攻击。
**提示词示例**：`身体左转背对目标，右腿伸直向后上方摆动，脚跟攻击对手面部，后摆腿`

### 6. 侧踹
**动作要领**：身体侧对目标，踢击腿屈膝提起，脚尖勾起外展，以脚外侧为力点侧向踹出。
**提示词示例**：`身体左转侧对目标，右腿屈膝提起脚尖外展，以脚外侧侧向踹出，右侧踹`

### 7. 鞭腿
**动作要领**：身体略转，踢击腿屈膝提起，以膝盖为轴小腿弹出，脚面绷直，弧形鞭打目标。
**提示词示例**：`身体左转，右腿屈膝提起，小腿弹出脚面绷直，弧形鞭打对手头部，右鞭腿`

### 8. 腾空侧踢
**动作要领**：双脚蹬地跳起，身体侧对目标，在空中完成侧踢动作。
**提示词示例**：`双脚蹬地跳起，身体右转侧对目标，左腿在空中直线蹬出，腾空左侧踢`

### 9. 二起腿
**动作要领**：第一腿踢出后不落地，直接在空中完成第二腿踢击。
**提示词示例**：`右腿踢出后不落地，直接在空中换左腿踢击，二起腿`

## 三、拳法类动作

### 1. 直拳
**动作要领**：拳头从下颌处直线向前击出，拳心向下，手臂伸直但不锁死，另一手护住下颌。
**提示词示例**：`右拳从下颌处直线向前击出，拳心向下，左手护住下颌，右直拳`

### 2. 勾拳
**动作要领**：手臂弯曲约90度，拳头从侧下方弧形向上击出，拳心向内，攻击下颌或腹部。
**提示词示例**：`右臂弯曲，拳头从侧下方弧形向上击出，攻击对手下颌，右勾拳`

### 3. 摆拳
**动作要领**：手臂弯曲约120度，拳头从侧面弧形摆击，拳心向下，攻击头部侧面。
**提示词示例**：`右臂弯曲，拳头从右侧弧形摆击对手头部左侧，右摆拳`

### 4. 刺拳
**动作要领**：前手快速直线击出，力量较轻但速度快，用于试探或干扰对手。
**提示词示例**：`左拳快速直线击出，用于试探干扰，左刺拳`

### 5. 翻背拳
**动作要领**：手臂伸直或微屈，拳头以肘为轴翻转，拳背由上向下或横向击打。
**提示词示例**：`右臂伸直，拳头以肘为轴翻转，拳背横向击打对手面部，翻背拳`

### 6. 贯拳
**动作要领**：手臂弯曲，拳头从侧面弧形向内击出，拳眼向上，攻击头部或躯干。
**提示词示例**：`右臂弯曲，拳头从右侧弧形向内击出，拳眼向上，右贯拳`

### 7. 劈拳
**动作要领**：手臂举过头顶，拳头由上向下劈击，拳眼向上，攻击头顶或锁骨。
**提示词示例**：`右臂举过头顶，拳头由上向下劈击对手头顶，右劈拳`

### 8. 日字冲拳
**动作要领**：肘部贴身，以肘底发力推动拳头沿身体中线直线冲出，拳眼朝上呈日字形，力达拳面，出拳与收拳同步进行，保持肘部始终贴近身体中线。
**提示词示例**：`右肘贴身，肘底发力推动右拳沿中线直线冲出，拳眼朝上呈日字形，左拳同步收回，日字冲拳`

## 四、肘膝类动作

### 1. 顶肘
**动作要领**：手臂弯曲握拳，肘尖向前直线顶出，另一手可推助发力。
**提示词示例**：`右臂弯曲握拳，肘尖向前直线顶出，左手推右拳助力，右顶肘`

### 2. 横肘
**动作要领**：手臂弯曲握拳，肘尖由外向内弧形摆击，攻击头部或躯干侧面。
**提示词示例**：`右臂弯曲，肘尖由外向内弧形摆击对手头部左侧，右横肘`

### 3. 挑肘
**动作要领**：手臂弯曲握拳，肘尖由下向上弧形挑击，攻击下颌或胸部。
**提示词示例**：`右臂弯曲，肘尖由下向上弧形挑击对手下颌，右挑肘`

### 4. 砸肘
**动作要领**：手臂弯曲握拳，肘尖由上向下砸击，攻击背部或头部。
**提示词示例**：`右臂弯曲，肘尖由上向下砸击对手后背，右砸肘`

### 5. 正膝
**动作要领**：单腿站立，另一腿屈膝提起，膝盖向前直线顶出，攻击腹部或胸部。
**提示词示例**：`左腿站立，右腿屈膝提起，膝盖向前直线顶出，攻击对手腹部，右正膝`

### 6. 侧膝
**动作要领**：身体侧转，屈膝腿从侧面弧形顶击，攻击大腿外侧或肋部。
**提示词示例**：`身体左转，右腿屈膝从侧面弧形顶击对手大腿外侧，右侧膝`

### 7. 飞膝
**动作要领**：双脚蹬地跳起，在空中屈膝顶出，攻击头部或躯干。
**提示词示例**：`双脚蹬地跳起，右膝在空中向前顶出，飞膝攻击`

## 五、闪避类动作

### 1. 下潜
**动作要领**：双膝弯曲，身体重心下降，头部和躯干向下潜躲，避开高位攻击。
**提示词示例**：`双膝弯曲，身体重心下降，头部向下潜躲对手直拳，下潜闪避`

### 2. 后仰
**动作要领**：上身后仰，重心后移，头部后撤，避开前向攻击。
**提示词示例**：`上身后仰重心后移，头部后撤避开对手摆拳，后仰闪避`

### 3. 侧闪
**动作要领**：头部和躯干向侧面移动，避开直线攻击，同时保持反击姿势。
**提示词示例**：`头部和躯干向左侧移动，避开对手直拳，左侧闪`

### 4. 摇避
**动作要领**：身体以腰部为轴做弧形摆动，头部呈U形轨迹移动，避开连续攻击。
**提示词示例**：`身体以腰部为轴弧形摆动，头部呈U形轨迹移动，摇避闪躲`

### 5. 格挡
**动作要领**：手臂抬起，以前臂或手掌向外或向内格开对手攻击。
**提示词示例**：`左臂抬起，以前臂向外格开对手直拳，左臂格挡`

### 6. 拍挡
**动作要领**：手掌快速向侧方拍击对手攻击的手臂或腿部，改变其攻击路线。
**提示词示例**：`左手向右侧拍击对手直拳，改变其攻击路线，左拍挡`

### 7. 接腿
**动作要领**：双手合抱接住对手踢击的腿部，为摔法或反击创造机会。
**提示词示例**：`双手合抱接住对手右腿，准备反击，接腿防守`

## 六、摔法类动作

### 1. 过肩摔
**动作要领**：身体背对对手，臀部顶住对手腹部，双手抓住对手手臂，弯腰发力将对手从肩上摔出。
**提示词示例**：`身体右转背对对手，臀部顶住对手腹部，双手抓住对手右臂，弯腰发力将对手从肩上摔出，过肩摔`

### 2. 绊摔
**动作要领**：一手推对手上身，一脚勾绊对手脚踝，上下合力将对手摔倒。
**提示词示例**：`左手推对手上身，右脚勾绊对手左脚踝，将对手摔倒，绊摔`

### 3. 抱腿摔
**动作要领**：双手抱住对手腿部，肩部顶住对手腹部，挺身发力将对手摔倒。
**提示词示例**：`双手抱住对手双腿，肩部顶住对手腹部，挺身发力将对手摔倒，抱双腿摔`

### 4. 夹颈摔
**动作要领**：手臂夹住对手颈部，身体侧转，以髋部为支点将对手摔倒。
**提示词示例**：`右臂夹住对手颈部，身体左转，以髋部为支点将对手摔倒，夹颈摔`

### 5. 抄腿摔
**动作要领**：一手抄起对手腿部，另一手推对手上身，转身发力将对手摔倒。
**提示词示例**：`左手抄起对手右腿，右手推对手上身，左转将对手摔倒，抄腿摔`

### 6. 抱腰摔
**动作要领**：双手抱住对手腰部，身体贴紧对手，挺身发力将对手向后摔倒。
**提示词示例**：`双手抱住对手腰部，身体贴紧对手，挺身发力将对手向后摔倒，抱腰摔`

## 七、组合动作示例

### 1. 拳腿组合
```
左刺拳试探→右直拳重击→左低扫腿
```
**提示词示例**：`左拳快速刺出试探，随即右拳直线重击对手面部，接左腿低扫踢击对手大腿外侧`

### 2. 肘膝组合
```
右直拳→左横肘→右正膝
```
**提示词示例**：`右拳直线击出，接左肘弧形摆击对手头部，随即右膝向前顶击对手腹部`

### 3. 攻防转换组合
```
左拍挡→右勾拳→左鞭腿
```
**提示词示例**：`左手拍开对手直拳，右拳弧形上勾对手下颌，接左腿弧形鞭打对手头部`

### 4. 闪避反击组合
```
下潜闪避→右上勾拳→右正膝
```
**提示词示例**：`身体下潜躲过对手摆拳，起身同时右拳上勾对手腹部，接右膝顶击`

### 5. 投摔组合
```
接腿→抄腿摔→地面压制
```
**提示词示例**：`双手接住对手右腿，左手抄起右腿右手推身，转身将对手摔倒，随即地面压制`

### 6. 高低组合
```
右直拳高位→变线低扫腿
```
**提示词示例**：`右拳向高位击出，对手防守时右腿迅速下压转为低扫踢击大腿外侧`

### 7. 连击组合
```
左刺拳→右直拳→左摆拳→右低扫
```
**提示词示例**：`左拳刺出，右拳直击，左拳弧形摆击头部，右腿低扫踢击大腿`
